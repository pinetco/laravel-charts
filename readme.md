## Installation

> Pull git repository in `packages` directory under project's root directory, and add below script in `composer.json` file

```json
"repositories": {
    "pinetco/charts": {
        "type": "path",
        "url": "/Users/{user}/code/{project}/packages/Pinetco/Charts"
    }
}
```

Via Composer

``` bash
$ composer require pinetco/charts
```
